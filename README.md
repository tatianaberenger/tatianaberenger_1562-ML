# Projeto de Machine Learning – Titanic

## Descrição do Problema
Este projeto tem como objetivo prever a sobrevivência de passageiros do Titanic a partir de características demográficas e socioeconômicas, configurando um problema de classificação binária supervisionada.

## Base de Dados
Foi utilizada a base pública do Titanic, contendo informações como classe do passageiro, sexo, idade, número de parentes a bordo, tarifa paga e porto de embarque.

## Metodologia
O projeto seguiu uma pipeline completa de Machine Learning, contemplando:
- Análise exploratória dos dados (EDA)
- Tratamento de valores ausentes
- Codificação de variáveis categóricas
- Padronização e redução de dimensionalidade
- Treinamento e validação de modelos
- Otimização de hiperparâmetros
- Avaliação com métricas de classificação e AUC
- Comparativo de desempenho entre modelos

## Resultados
Os modelos apresentaram desempenho satisfatório, com destaque para a Regressão Logística e Random Forest. A métrica AUC foi utilizada como principal critério de comparação.
